# Reglas de seguridad

1. La aplicación es exclusivamente formativa y no es enseñanza oficial de Amadeus.
2. No se conecta a sistemas reales ni contiene credenciales, endpoints o SDK de proveedores.
3. Solo se utilizan pasajeros, contactos, empresas, localizadores, vuelos, precios y políticas ficticios.
4. No se deben introducir datos reales de clientes, empleados o viajeros.
5. Las operaciones de emisión, pago, reemisión, reembolso, void y cancelación de proveedor están bloqueadas.
6. Cancelar o modificar dentro del simulador solo cambia el estado local del navegador.
7. Los códigos públicos de aerolínea, aeropuerto, cadena u operador se usan únicamente como referencias estáticas.
8. Los procedimientos reales deben comprobarse en formación oficial y en documentación interna autorizada.
9. Las políticas, colas, perfiles y autorizaciones de AMEX GBT no se inventan; solo podrán incorporarse desde material autorizado.
10. Ante una situación ambigua, el contenido debe enseñar a detenerse, verificar y escalar.
