# Product brief

## Product name

Amadeus Learning Coach

## User

A beginner Travel Consultant learning Amadeus GDS in a corporate travel context.

The learner is not yet experienced with Amadeus.

## Problem

Learning Amadeus is difficult because beginners often jump directly into commands without understanding the underlying concepts.

That is dangerous because real GDS actions can affect PNRs, ticketing, refunds, exchanges, queues, and customer travel.

## Solution

Build a safe educational app that teaches Amadeus-related concepts progressively and includes a fake terminal simulator.

The app must focus on understanding before command practice.

## Learning model

Each lesson should include:
1. Objective
2. Explanation
3. Example
4. Exercise
5. Correction criteria
6. Mini quiz
7. Summary
8. Next step

## Course duration

6 weeks.

5 study days per week.

30 lessons total.

Each day should take 30-45 minutes.

## Main modules

1. Week 1: Basic GDS and travel concepts
2. Week 2: PNR anatomy and reservation logic
3. Week 3: SSR, OSI, seats, baggage, special services
4. Week 4: Fares, fare rules, policy, economic risk
5. Week 5: Changes, refunds, voids, reissues, queues
6. Week 6: Progressive simulations and final assessment

## Terminal simulator

The terminal is not a real Amadeus terminal.

It should be a safe learning environment.

It can accept fictional commands such as:

- HELP
- GLOSSARY PNR
- SHOW SAMPLE_PNR
- SHOW AVAILABILITY MAD AMS
- SHOW FARE_RULE BASIC
- PRACTICE SEGMENTS
- PRACTICE SSR_OSI
- RESET

The simulator should explain what the learner is seeing.

It should not claim these are exact real Amadeus commands.

## Safety

Add persistent warnings:
- Training only
- No real bookings
- Verify real workflows with official training and internal procedures
- Do not enter real customer data

## Success criteria

A beginner can use the app to:
- understand core concepts
- practice identifying risks
- distinguish PNR from ticket
- understand SSR vs OSI
- reason about fares
- know when to escalate
- practice in a fake terminal safely
